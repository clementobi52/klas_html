Secondary Application Process

Steps for processing unit owner applications

1. Access Application

Unit owner receives invitation link or searches for their property using the property code

2. Unit Selection

Applicant selects or confirms their specific unit within the property

3. Document Submission

Upload proof of purchase, identity documents, and other supporting documentation

4. Verification and Approval

Department verifies documents and approves the secondary application

5. CofO Issuance

Individual Certificate of Occupancy is generated for the specific unit
